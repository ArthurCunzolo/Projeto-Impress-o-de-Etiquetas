# 📄 Impressão de Envelopes DL

Neste repositório foi desenvolvida, com auxílio de Inteligência Artificial, uma página web que facilita a impressão de envelopes no formato **DL**.

O objetivo da aplicação é tornar mais prática a impressão dos dados de:

- 📤 **Remetente**
- 📥 **Destinatários (clientes)**

A ferramenta permite cadastrar os dados de forma simples e gerar arquivos prontos para impressão, otimizando o processo e evitando preenchimentos manuais.

## 🚀 Funcionalidades

- Cadastro de **remetente**
- Adição de **múltiplos destinatários**
- Geração automática de **PDFs prontos para impressão**
- Layout ajustado para **envelopes padrão DL**
- Preenchimento automático de endereço via **CEP**

## 💡 Objetivo

Automatizar e agilizar o processo de impressão de envelopes utilizados em correspondências empresariais, reduzindo erros e aumentando a produtividade.
